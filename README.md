# FluMinGo

---

## 概要

FluMinGoは、Flutterを使用したフロントエンドとGolangを使用したバックエンドを組み合わせた業務用アプリケーション開発フレームワークです。  
このフレームワークは、高速な開発サイクル、スケーラブルなアーキテクチャ、そして高い保守性を提供します。
