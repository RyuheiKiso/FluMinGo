# デプロイ手順

### フロントエンド
1. ビルドを実行します。
    ```bash
    flutter build apk
    ```

### バックエンド
1. バイナリをビルドします。
    ```bash
    go build -o flumingo
    ```

デプロイ後の確認手順については、各技術の公式ドキュメントを参照してください。
